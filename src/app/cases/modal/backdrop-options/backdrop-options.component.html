Default Backdrop This modal has the default backdrop with semi-transparent black background. You can click outside the modal (on the backdrop) to close it. This is the standard behavior.
No Backdrop This modal has no backdrop, so you can see the content behind it. You can only close this modal using the close button in the header, not by clicking outside.
Custom Backdrop This modal has a custom blue backdrop with 30% opacity. Notice the blue tint on the backdrop. You cannot close this modal by clicking the backdrop.
No Click to Close This modal has a backdrop but clicking on it won't close the modal. You can only close this modal using the close button in the header. This is useful for important dialogs where you want to prevent accidental closure.
