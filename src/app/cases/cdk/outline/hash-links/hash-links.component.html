Hash Links Example

What are Hash Links?

Hash links are clickable anchor symbols (#) that appear next to each section heading. They allow users to copy the URL to a specific section or navigate directly to it.

How to Enable

To enable hash links, set the showHashItem property to true on the axOutlineContainer directive. You can also enable URL copying by setting copyUrlOnClick to true.

User Experience

When users hover over a section heading, a hash link appears. Clicking it will either copy the URL to the clipboard (if copyUrlOnClick is enabled) or navigate to that section.

Accessibility

Hash links are marked with aria-hidden="true" to avoid confusion for screen readers, as the main heading text already provides the necessary information.

Styling

The hash links are styled to be subtle and only appear on hover, ensuring they don't interfere with the reading experience while remaining easily accessible when needed.
