Outline Directive Usage

Introduction

The outline directive provides a way to create dynamic navigation menus that automatically highlight the currently visible section as you scroll through the content.

Features

This directive offers several powerful features including automatic item detection, scroll-based activation, and smooth scrolling navigation.

Automatic Detection

The directive automatically detects elements marked with the axOutlineItem directive and creates a navigation structure based on their hierarchy.

Scroll Activation

As you scroll through the content, the corresponding navigation item becomes active, providing visual feedback about your current position in the document.

Implementation

To use the outline directive, simply apply the axOutlineContainer directive to a container element and mark the sections you want to include in the navigation with axOutlineItem.

Benefits

This approach provides better user experience for long-form content, making it easier for users to navigate and understand their current position within the document.
