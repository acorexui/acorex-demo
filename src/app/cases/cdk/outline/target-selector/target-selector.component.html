Target Selector Example

Main Sections

This example demonstrates the target selector functionality. The outline container is configured to target h3 elements specifically, so only h3 headings will appear in the outline navigation.

How Target Selector Works

The target selector allows you to specify which elements should be included in the outline navigation using a CSS selector. This is useful when you want to include elements that don't have the axOutlineItem directive.

Combining with axOutlineItem

You can combine the target selector with explicit axOutlineItem directives. Elements with axOutlineItem will always be included, and the target selector will add additional elements that match the selector.
