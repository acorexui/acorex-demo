Scroll Offset Example

What is Activation Offset?

The activation offset determines how far from the top of the viewport an outline item needs to be before it becomes active. This example uses a 100px offset, meaning items become active when they're 100 pixels from the top of the viewport.

Default vs Custom Offset

By default, the activation offset is 16px. In this example, we've increased it to 100px to demonstrate how it affects the timing of when outline items become active.

Use Cases

A larger offset is useful when you have fixed headers or navigation bars that might cover the content, ensuring the outline activates at the right moment for your layout.

Responsive Considerations

Consider adjusting the activation offset based on your responsive design. You might want different offsets for mobile vs desktop layouts to account for different header heights.

Performance Impact

The activation offset doesn't significantly impact performance, but it does affect the user experience by determining when navigation items become active during scrolling.
